## Description

diablo4.trading backend application

## Windows Installation
Follow this setup guide - [Windows Setup Instructions](https://github.com/SanctuaryTeam/.github/blob/main/profile/SETUP.md)

## Unix Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
